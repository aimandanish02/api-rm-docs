### Response Parameters

:::note

- Current type in <b>Status</b> will be :-

  - "SUCCESS" <br/>
  - "MYKAD_NOT_FOUND" <br/>
  - "MYKAD_SINGLE_NOT_FOUND" <br/>
  - "MYKAD_FACE_NOT_FOUND" <br/>
  - "INCOMPLETE" <br/>
  - "OCR_ENGINE_ERROR" <br/>
  - "OCR_ENGINE_CONNECTION" <br/>

- Current type in <b>Action</b> will be :-
  - "NONE_REQUIRED"<br/>
  - "ATTENTION_REQUIRED"<br/>
  - "INCONCLUSIVE"<br/>
  - "RESOLVED"<br/>

:::
