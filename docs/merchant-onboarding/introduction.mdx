# Merchant Onboarding API

:::note
This API is for **Partner Merchant accounts** only. To activate a Partner account, [contact us](https://revenuemonster.my/about-us).
:::

Onboarding API is for third-party developers who have their own merchant base and want to integrate RM's Open API. Merchants are onboarded directly through the partner, so the partner collects and submits the merchant information on their behalf.

![Merchant Onboarding flow](/img/onboarding/onboarding.png)

:::note
For all Open API endpoints other than the Onboarding API, you must use an **Onboarded Merchant Account** Developer Application. See the configuration guide below.
:::
